# Sales Summary
The objective of this project is to provide the retail store owner with insights to enable them quickly identify their hottest leads, gain deeper insight into buyer behavior, and drive engagements that close deals.

# About the Data
The dataset was gotten from [Kaggle](https://www.kaggle.com/kyanyoga/sample-sales-data?select=sales_data_sample.csv). It contains the following information, Order Info, Sales, Customer, Shipping, City, Country, Territory, Postal Code, Deal Size, etc.
